# 2Dots Hires Website

**2Dots Hires** is a hiring website for a fictional logistics & delivery company 2Dots, built for learning HTML and frontend practices. It demonstrates forms, tables, semantic tags, and multi-page navigation in a realistic website layout.

[Website Link](https://project2-2-dots-hires.vercel.app/)

## 📁 Project Structure

```text
Project-2Dots-Hires/
│
├── images/             # Project assets and logos
├── index.html          # Home page
├── knowus.html         # About page
├── jobs.html           # Job listings
└── apply.html          # Application form
```
## Pages Overview

1. **Home Page** – Hero section, company intro, mission & vision, services, and team.  
2. **Know Us** – Company information, values, and philosophy.  
3. **Jobs** – Open positions, job descriptions, hiring process table.  
4. **Apply** – Application form with inputs: text, email, number, radio, checkbox, select, textarea.

## Key Features

- Semantic HTML5 tags (`header`, `nav`, `main`, `section`, `footer`)  
- Forms with proper **labels, fieldsets, legends, and validation**  
- Tables using **caption, thead, tbody, tfoot, rowspan, colspan**  
- Multi-page navigation and professional layout  
- Educational focus for practicing HTML concepts
